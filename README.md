# Tandoor API Automation Project

## Description
Automated API testing project for Tandoor Recipes application.

## Tech stack
- Python 3.13
- pytest
- requests
- python-dotenv

## How to run

1. Install dependencies:
pip install -r requirements.txt

2. Create .env file:
BASE_URL=https://app.tandoor.dev
TANDOOR_TOKEN=your_token_here

3. Run tests:
pytest -v
